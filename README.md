# darwin_slp_fda
Description: Functional Data Analysis of Darwin sea level pressure (1951-2025). Fourier smoothing, FPCA &amp; VARIMAX rotation identify seasonal patterns &amp; ENSO signals. First 2 PCs explain 79.7% variation - separating summer ENSO from winter monsoon signals.  Tags: functional-data-analysis, fda, darwin-slp, enso, climatology, pca, varimax, rstats
